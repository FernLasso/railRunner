    RailRunners

    RailRunners is a website meant to help anyone who is traveling with public transport, by providing them with real-time data about the whereabouts of their transportation vehicle. The uniqueness of RailRunenrs comes into light with the abiliy to offer alternative travel advice that can spare you a lot of time. And as we all know, time is money. Railrunners therefore created an algorithm that calculates alternatives with the option of walking faster than normally is expected. As common rail runners, we know that a bit of running can change your day.

    API Usage

    RailRunner.com uses the OTP (OpenTravelPlanner) API to collect the information needed to present you with the most accurate data possible.
